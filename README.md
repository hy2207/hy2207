## 🧩 Selected Projects

### 📚 Natural Language Book Search PoC

Built a natural-language book search and recommendation PoC using **GPT API**, **function calling**, **Text-to-SQL**, **FastAPI**, and **SQLite**. The goal was to ground LLM responses in actual database results instead of relying on free-form generation.

### 🎧 E-book TTS Preprocessing

Developed preprocessing workflows for EPUB/XHTML content to make e-book text cleaner and more suitable for TTS input. The work included HTML parsing, sentence cleanup, number and symbol handling, pause rules, and rule-based exception management.

### 📰 News and Social Text Analytics Pipeline

Processed crawled news and online text data, connected language model analysis results, and organized outputs into **MySQL** for service-side search and retrieval. Improved data structures for keyword search and large-scale query workflows.

### 📈 Supply Chain Risk Monitoring

Created structured risk-monitoring data from news articles using **NER**, **event extraction**, risk scoring, **JSON**, **SQLite**, and dashboard-ready data formats.

### 🤖 Robot Assistant LLM / RAG

Explored LLM and RAG workflows for a robot assistant, including natural-language command conversion, user context retrieval, embedding, vector DB review, reranking, and edge/local model feasibility.

### 🔐 Secure Environment NLP PoC

Compared GPT API, local LLM, ONNX/BERT-style approaches, and LAC-based NLP under security, cost, speed, and CPU-only constraints. Helped review a more stable NLP direction for constrained operational environments.

### 🚆 Railway Operations Data and Automation

Analyzed operational documents and data to review automation feasibility. Work included PDF/XML parsing, rule-based logic review, clustering or ML/LLM applicability, feasibility documentation, open item tracking, and PM coordination.

### 📝 AI R&D PM Documentation

Supported AI R&D collaboration through meeting preparation, meeting minutes, action item tracking, risk and decision logs, KPI discussion, acceptance criteria, stakeholder communication, and Codex-based PM documentation workflows.
